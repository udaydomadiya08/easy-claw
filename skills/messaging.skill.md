# messaging
Send messages via any connected channel

## Trigger
When user wants to send a message, notification, or alert

## Instructions
1. Determine target channel (last used or specified)
2. Determine target recipient
3. Use: `openclaw message send --channel <channel> --target <recipient> --message "<content>"`
4. Confirm delivery

## Channels
- telegram, discord, slack, whatsapp, signal, imessage
